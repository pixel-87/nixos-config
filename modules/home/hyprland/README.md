# Hyprland Keybindings

This document lists the configured keybindings for the Hyprland window manager setup.

**Mod Key**: `SUPER` (Windows Key)

## General
| Keybinding | Action |
|------------|--------|
| `SUPER + F1` | Show Keybinds |
| `SUPER + Q` | Open Terminal (Kitty) |
| `SUPER + C` | Kill Active Window |
| `SUPER + M` | Exit Hyprland |
| `SUPER + E` | Open File Manager (Nautilus) |
| `SUPER + V` | Toggle Floating Window |
| `SUPER + R` | Open App Launcher (Wofi) |
| `SUPER + P` | Toggle Pseudo Tiling |
| `SUPER + X` | Toggle Split |
| `SUPER + F` | Open Browser (Zen) |

## Window Management & Navigation
| Keybinding | Action |
|------------|--------|
| `SUPER + ←/→/↑/↓` | Move Focus |
| `SUPER + H/J/K/L` | Move Focus (Vim keys) |
| `SUPER + SHIFT + ←/→/↑/↓` | Move Window Position |
| `SUPER + SHIFT + H/J/K/L` | Move Window Position (Vim keys) |
| `CTRL + ALT + ↑` | Focus Floating Window |
| `CTRL + ALT + ↓` | Focus Tiled Window |

## Workspaces
| Keybinding | Action |
|------------|--------|
| `SUPER + [1-9]` | Switch to Workspace [1-9] |
| `SUPER + SHIFT + [1-9]` | Move Window to Workspace [1-9] (Silent) |
| `SUPER + S` | Toggle Special Workspace (Magic) |
| `SUPER + SHIFT + S` | Move Window to Special Workspace |
| `SUPER + CTRL + C` | Move Window to New Empty Workspace |
| `SUPER + Mouse Scroll` | Cycle Through Workspaces |

## Screenshots
| Keybinding | Action |
|------------|--------|
| `SUPER + SHIFT + P` | Screenshot Area |
| `SUPER + CTRL + P` | Screenshot Window |
| `SUPER + ALT + P` | Screenshot & Edit |

## Wallpaper Control
| Keybinding | Action |
|------------|--------|
| `SUPER + ALT + W` | Open Wallpaper Picker |
| `SUPER + CTRL + →` | Next Wallpaper |
| `SUPER + CTRL + ←` | Previous Wallpaper |

## Media & Hardware
| Keybinding | Action |
|------------|--------|
| `Audio Mute` | Toggle Audio Mute |
| `Mic Mute` | Toggle Microphone Mute |
| `Brightness Up/Down` | Adjust Screen Brightness |
| `Volume Up/Down` | Adjust Volume |
| `Media Play/Pause` | Play/Pause |
| `Media Next/Prev` | Next/Previous Track |

## Mouse Bindings
| Keybinding | Action |
|------------|--------|
| `SUPER + Left Click` | Move Window |
| `SUPER + Right Click` | Resize Window |
